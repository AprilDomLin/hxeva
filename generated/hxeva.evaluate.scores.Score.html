Score¶

class hxeva.evaluate.scores.Score[源代码]¶

评分指标计算类。

注解

可以直接通过调用 calc 和 scores 方法计算多等级、多类型的评分结果。

classmethod acc(tables)[源代码]¶

准确率 / 晴雨评分

\(Accuracy = \frac{tp + tn}{tp + fp + fn + tn}\)

参数

tables (hxeva.evaluate.scores.ContingencyTable) – 联列表分类结果，包含 tp 、 fp 、 fn 和 tn 的数值。

返回

准确率 / 晴雨评分。当预报无降水时或未对应等级降水时，返回 NaN 。

classmethod calc(truth, test, **kwargs)[源代码]¶

算法人员调用进行相关评分计算

参数

  • truth (numpy.ndarray) – 真实值数组

  • test (numpy.ndarray) – 待测试值数组

关键字参数

  • threshold (list) – 默认为 [0, 0.1, 2.5, 8.0, 16.0, 150]。 分类阈值表，阈值列表最小值和最大值以外的值，认为是异常值，不纳入计算。

  • level (list[int]) – 默认为 [0, 1, ... len(threshold) - 1]。 需要计算评分的等级。

  • names (list[str]) – 默认为 ["TS", "PO", "FAR", "POD", "ACC", "HSS"]。需要计算的评分内容。

  • method (str, optional) – 默认为 binary。参数可选择 binary (二分类), multi (多分类) 和 grade (多级)。

  • window_size (tuple) – 默认为 None。领域匹配半径格点数目。

  • acc (int) – 默认为 4。计算结果的精度值，即保留有效小数点位数

  • save_path (str) – 默认为 ./scores.csv。评分结果csv文件的保存路径。

classmethod scores(truth, test, **kwargs)[源代码]¶

更灵活的评分调用方式，类似calc。但没有封装结果保存与打印。

参数

  • truth (numpy.ndarray) – 真实值数组

  • test (numpy.ndarray) – 待测试值数组

关键字参数

  • threshold (list) – 默认为 [0, 0.1, 2.5, 8.0, 16.0, 150]。 分类阈值表，阈值列表最小值和最大值以外的值，认为是异常值，不纳入计算。

  • level (list[int]) – 默认为 [0, 1, ... len(threshold) - 1]。 需要计算评分的等级。

  • names (list[str]) – 默认为 ["TS", "PO", "FAR", "POD", "ACC", "HSS"]。需要计算的评分内容。

  • method (str, optional) – 默认为 binary。参数可选择 binary (二分类), multi (多分类) 和 grade (多级)。

  • window_size (tuple) – 默认为 None。领域匹配半径格点数目。

  • acc (int) – 默认为 4。计算结果的精度值，即保留有效小数点位数

返回

各等级的不同评分结果。
