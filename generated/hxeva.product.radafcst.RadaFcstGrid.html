RadaFcstGrid¶

class hxeva.product.radafcst.RadaFcstGrid(name='RADA_PRE_HOR_03H_GRID', extend=None, **kwargs)[源代码]¶

雷达短时预报产品格点检验评估

参数

  • name (str) – 默认为 RADA_PRE_HOR_03H_GRID。产品名称。

  • extend (Optional[dict]) – 默认为 None。经纬度范围。

关键字参数

  • log_root – 默认为 ./logs。日志文件夹根目录

  • max_counter – 默认为 10。文件错误最大循环等待次数

  • shape – 默认为 None。评估数据形状大小

evaluate(tag_time, truth_file, test_file, **kwargs)[源代码]¶

雷达短时预报产品格点检验评估

参数

  • tag_time (datetime.datetime) – 当前时间

  • truth_file (dict) – 真值数据信息。参数信息关键字参考CMAE config。

  • test_file (dict) – 待测数据信息。参数信息关键字参考CMAE config。

关键字参数

  • save (dict) – 检验评估产品保存路径规则。

  • timefcst (list) – 预报时长表。涵盖预报产品所有的预报时长的列表。

  • key (list) – 默认为 PRE_1h。检验评估的变量关键字。

  • threshold (list) – 默认为 [0, 0.1, 2.5, 8.0, 16.0, 150]。雷达回波阈值表。

  • location (dict or None) – 默认为 None。检验评估经纬度范围。

  • code (str) – 默认为 RADA_PRE_HOR_03H_GRID。检验评估对象的名称或代号。

  • area (str) – 默认为 None。 检验区域名称，检验评估产品字段。

返回

是否正常完成评估。True 正常完成评估; False 未正常完成评估，或因路径错误。
