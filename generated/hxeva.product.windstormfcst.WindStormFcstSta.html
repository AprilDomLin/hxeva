WindStormFcstSta¶

class hxeva.product.windstormfcst.WindStormFcstSta(name='WindStorm', log_root='./logs', extend=None)[源代码]¶

雷暴大风概率预报产品站点检验评估

参数

  • name – 默认为 WindStorm。产品名称。

  • log_root – 默认为 ./logs 。日志文件夹根目录

  • extend – 默认为 None 。经纬度范围。

evaluate(tag_time, truth_file, test_file, **kwargs)[源代码]¶

雷暴大风概率预报产品站点检验评估

参数

  • tag_time (datetime.datetime) – 当前时间

  • truth_file (dict) – 真值数据信息。参数信息关键字参考CMAE config。

  • test_file (dict) – 待测数据信息。参数信息关键字参考CMAE config。

关键字参数

  • file (str) – 检验评估产品保存路径规则。

  • timefcst (list) – 预报时长表。涵盖预报产品所有的预报时长的列表。

  • key (list) – 默认为 Windstorm。检验评估的变量关键字。

  • threshold (list) – 默认为 [0, 50, 101]。雷暴大风概率预报阈值表。

  • location (dict or None) – 默认为 None。检验评估经纬度范围。

  • code (str) – 默认为 WINDSTORM_03H_STATION。检验评估对象的名称或代号。

  • max_counter (int) – 默认为 10。 数据产品检验评估错误时，循环等待的次数。

  • area (str) – 默认为 None。 检验区域名称，检验评估产品字段。

返回

是否正常完成评估。True 正常完成评估; False 未正常完成评估，或因路径错误。
