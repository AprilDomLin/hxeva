业务检验基类

包含业务日志、实时业务逻辑、时区转化等基础功能。

默认为 {log_root}/{name}。保存日志的日志文件夹。

Type

Path

logger¶

业务日志记录器。

Type

logging.Logger

counter¶

默认为 0。计数器。用于业务运行时，判断是否需要忽略错误，生产缺失检验结果。

Type

int

参数

  • logger (logging.Logger) – 默认为 None。业务日志记录器。

  • name (str) – 默认为 空。产品名称。

  • log_root (str) – 默认为 ./logs。日志文件夹根目录

产品评估检验

  • tag_time (datetime.datetime) – 当前时间

  • truth_file (dict) – 真值数据信息。参数信息关键字参考CMAE config。

返回

main(tag_time=None, **kwargs)[源代码]¶

实时业务运行主程序

参数

tag_time (Optional[datetime.datetime]) – 当前时间戳。

关键字参数

  • fmt (str) – 默认为 %Y%m%d%H。日志文件保存的时间格式。

  • days (int) – 默认为 7。 日志保存的时长，单位为: day。

  • sleep_time (int) – 默认为 60。 当数据出现错误，循环等待的时间，单位为: s。

  • delay (timedelta) – 默认为 20。 根据tag_time，延迟检验的时间，单位为: min。

static transform_time(time, timezone, rollback=False)[源代码]¶

时区转化

当前仅支持基于世界时(UTC)的转化和逆转化。

参数

  • time (datetime.datetime) – 时间戳，UTC。

  • timezone (str) – 时区名称

  • rollback (bool) – 默认为 False。是否由 timezone 回滚为世界时。True, 逆转化为UTC。False, 转化为 timezone。

返回

转化为 timezone 的时间
